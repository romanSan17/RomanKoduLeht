MVP

Minimal Viable Product - toote, teenuse või teenuse testversioon, millel on minimaalne hulk funktsioone (mõnikord isegi üks), mis kannab lõppkasutajale väärtust. Võite kasutada MVP-d oma projektides oma idee kasulikkuse testimiseks. Kas publikule on seda vaja ilma algselt kogu raha investeerimata.

GIT remote

Käsuga git remote saate luua, vaadata ja kustutada ühendusi teiste repositooriumidega. Kaugühendused on pigem nagu järjehoidjad kui otsesed lingid teistesse repositooriumidesse.
  1. Mine saidile, loo uus repositoorium ja saada selle kloonimise URL aadress
  2. Kohaliku repositooriumi initsialiseerimine:
    git init
    git add .
    git commit -m "Initial commit"
  3. Ühendamine kaugrepositooriumiga: git remote add origin https://github.com/username/repository-name.git
  4. Muudatuste saatmine kaugrepositooriumi
